# What is this?
This python program highlights key when you press one, best used with obs
## Config (OBS)
### Note: keycrop.txt will only be created when you first launch the exe
Use **keycrop.txt** to crop the keys correctly
Example **keycrop.txt**:
```
KEY "Q":
Start position: 10
End position: 285
KEY "E":
Start position: 235
End position: 21
```
## Customization
You *are* able to customize the display names of the keys, but **more** customization is coming soon *;)*
